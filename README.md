# 小区物业管理系统 java104

## 项目概述

小区物业管理系统是一款基于Java开发，采用前后端分离、分布式架构设计的社区管理系统。它支持移动端访问，能够有效提高物业管理效率，增强社区稳定性，促进社区经济循环。

## 技术栈

### 后端

- **Spring Boot**：核心后端框架
- **MyBatis**：数据持久化框架
- **Redis**：缓存和会话管理

### 前端

- **Vue.js**：前端框架
- **Element UI**：前端UI框架

## 功能模块

- **收费+报表**：自动化处理物业费用收取和财务报表生成
- **报修系统**：线上报修，快速响应与处理
- **门禁道闸**：设备管理，出入口控制
- **线上缴费**：移动端便捷缴费
- **社区商圈**：整合社区商业资源
- **巡检功能**：物业日常巡检
- **信息推送**：欠费通知与社区公告推送

## 系统角色

- 住户：报修、缴费、接收通知等
- 物业人员：收费、巡检、发布公告等
- 商户：商圈管理

## 运行环境

- 后端：JDK1.8+，支持MySQL、Redis的环境
- 前端：支持现代浏览器的环境

## 部署步骤

1. 部署后端服务
   - 安装JDK1.8+
   - 部署MySQL数据库，导入数据
   - 部署Redis服务
   - 部署Spring Boot应用

2. 部署前端服务
   - 使用Vue.js构建前端工程
   - 部署静态资源到Web服务器
